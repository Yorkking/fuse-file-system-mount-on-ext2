# 基于 fuse 的多级调度文件系统

下面介绍一下该目录下的各种文件。

## 核心文件

main.c: 实现 fuse 提供的文件接口

util.h: 定义了存在 DRAM 的目录多叉树的数据结构
util.c: 实现上述结构的各种操作 

control.c, control.h: 实现调度算法

目前已经完成了持久化到 pmem 的操作，同时也完成了 pmem 和 disk 的动态调度的过程。
